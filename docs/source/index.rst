Welcome to SmartOven's documentation!
=====================================

SmartOven is an oven, but smart. 
This is an IoT project which aims to make baking a simpler and more enjoyable experience.

Check out the :doc:`Installation and Usage <content/installation_and_usage>` section for information about how to install and use this project.

After that, you can check out the :doc:`API docs and Features <content/api_docs_and_features>` provided by this application.

.. note::

   This project is under active development.
